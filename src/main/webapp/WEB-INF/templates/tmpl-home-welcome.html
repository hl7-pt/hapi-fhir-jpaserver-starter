Este servidor foi instanciado pelo chapter HL7 Portugal para ajudar a implementação e testes de FHIR em Portugal. Esta implementação da especificação FHIR usa uma stack totalmente open source.

Este servidor é contruido sobre um numero de módulos open source como o projecto HAPI FHIR que é uma implementação 100% open-source (Apache 2.0 Licensed) da especficação FHIR.

Este Servidor é DE TESTES e não deve ser usado para fins de produção. Além disso não deve ser usado com dados reais, potencialmente sensíveis, ou dados de pacientes. Para qualquer dúvida, enviar email para tech[@]hl7 dot pt
